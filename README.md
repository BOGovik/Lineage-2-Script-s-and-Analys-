# Lineage-2-Script-s-and-Analys-
Save Money Bot — это продвинутый программно-аппаратный комплекс для автоматизации действий пользователя. Проект использует связку Python + Arduino, что позволяет имитировать ввод данных на уровне «железа» (HID). Это делает бота практически невидимым для систем защиты, так как компьютер распознает его как реальную физическую мышь и клавиатуру.
